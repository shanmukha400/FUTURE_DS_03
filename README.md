# Bank Marketing Funnel & Conversion Analysis
## Project Overview

This project analyzes a bank marketing campaign dataset to understand customer behavior and conversion patterns.

The goal is to identify how customers move through the marketing funnel and what factors influence subscription decisions.

## Objective
Analyze customer conversion funnel

Identify drop-off points in marketing stages

Understand factors affecting subscription

Improve marketing efficiency using data insights
## Dataset

Bank Marketing Dataset (UCI Repository)

https://archive.ics.uci.edu/ml/datasets/bank+marketing

## Tools Used
Python

Pandas

NumPy

Matplotlib

Seaborn

Google Colab
## Workflow
### 1. Data Cleaning
Handled missing values

Removed duplicates

Converted target variable into numerical format
### 2. Exploratory Data Analysis (EDA)
Customer demographic analysis

Campaign performance analysis

Conversion behavior study
### 3. Funnel Analysis
Contacted → Engaged → Subscribed stages

Drop-off identification
### 4. Visualization & Metrics
Funnel chart → shows drop-off (Contacted → Interested → Subscribed)

Barh job plot → job-wise conversion differences

Boxplot → age impact on subscription

Line plot → contact type performance

Countplot → marital status impact

Pie chart → overall conversion ratio

Scatter plot → age vs duration relationship

Histogram → distribution of call duration
## Key Insights
Overall conversion rate is low compared to total contacts

Call duration is the strongest factor influencing subscription

Middle-aged customers show higher conversion rates

Certain job categories perform better in campaigns
## Business Recommendations
Improve call quality instead of increasing call volume

Focus on high-intent customers with longer engagement

Target high-performing job and age segments

Reduce unnecessary repeated campaign calls
## Outcome

This analysis helps improve:

Marketing efficiency

Customer targeting strategy

Conversion rate optimization

Campaign cost reduction
